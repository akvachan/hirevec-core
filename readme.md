## Features
- [ ] No registration
- [ ] No ads 
- [ ] No manual filters
- [ ] No premium and/or paywall
- [ ] Offline access
- [ ] Free and open source 
- [ ] Strong anti-spam for matching, chatting and posting
- [ ] Chat directly after a match
- [ ] Zero glitch UI 
- [ ] Maximum 69 swipes per day
- [ ] Basic job search statistics dashboard
- [ ] See a potential match
- [ ] Best positions served first
- [ ] Ability to choose a visual theme / appearance of the app
- [ ] Ability to customize position/candidate profile cards
- [ ] Basic profile creation
- [ ] Instant profile creation via CV upload 
- [ ] Hard deletion of user data in a single click 
- [ ] GDPR compliance
- [ ] Anonymized company and candidate profiles
- [ ] No ability to lookup profiles, candidates or positions manually
